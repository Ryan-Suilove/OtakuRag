# 欢迎来到实力至上主义的教室 (Classroom of the Elite) - RAG 知识库文档

## 基础信息
- **中文名**：欢迎来到实力至上主义的教室
- **日文名**：ようこそ実力至上主義の教室へ (Yōkoso Jitsuryoku Shijō Shugi no Kyōshitsu e)
- **类型**：校园、心理博弈、悬疑、智斗、轻小说改编
- **首播年份**：2017 年（第一季）
- **制作公司**：Lerche（第一、二季），Studio Kai（第三季）
- **原作**：衣笠彰梧（著作）、知世俊作（插画）的轻小说系列

## 主要角色

### 角色名：绫小路清隆 (Ayanokoji Kiyotaka)
- **身份**：主角 / D 班幕后操纵者
- **声优**：千叶翔也
- **人物设定**：高度育成高中 D 班学生，外表冷漠平凡，刻意隐藏实力。实为“白色房间”培养出的完美人造天才，拥有极高的智商、体能及心理操控能力。他将他人视为工具，追求平淡生活却不断被卷入纷争，擅长在幕后布局操控全局。
- **经典台词**：“所有的人类都是工具。无论怎么使用，都取决于使用者。”

### 角色名：堀北铃音 (Horikita Suzune)
- **身份**：女主角之一 / D 班明面领导者（初期）
- **声优**：鬼头明里
- **人物设定**：性格孤高冷傲，拒绝与人交流，一心只想晋升至 A 班以得到哥哥认可。初期缺乏领导力且不善交际，但在绫小路的暗中引导和自身经历挫折后，逐渐学会信任同伴并成长为能够统率班级的领袖人物，是绫小路在明面上的“面具”。
- **经典台词**：“我不需要朋友。只要能达到目的，我一个人就够了。”

### 角色名：轻井泽惠 (Karuizawa Kei)
- **身份**：D 班女生领袖 / 绫小路的合作者（后期）
- **声优**：竹达彩奈
- **人物设定**：表面是 gyaru（辣妹）风格、依附强者的现充女生，实则因初中遭受严重霸凌而内心脆弱，通过伪装强势来保护自己。被绫小路识破秘密后成为其专属“棋子”，在多次特别考试中发挥关键作用，后期对绫小路产生深厚依赖与情感。
- **经典台词**：“如果是你的话，无论什么命令我都会听。”

### 角色名：坂柳有栖 (Sakayanagi Arisu)
- **身份**：A 班领袖 / 主要竞争对手
- **声优**：日高里菜
- **人物设定**：A 班核心人物，身患先天疾病需坐轮椅，但拥有天才般的头脑和优雅从容的气质。性格腹黑且享受智力游戏，视绫小路为唯一的“特别存在”和劲敌。她擅长利用人心和规则漏洞，是以正面交锋和策略布局著称的强敌。
- **经典台词**：“啊啦，真是有趣。能让我如此兴奋的对手，除了你之外别无二人。”

## 剧情概要

- **一句话简介**：在一所实行绝对实力主义的政府寄宿高中里，看似最弱的 D 班学生在神秘天才少年的幕后操控下，通过一系列残酷的特别考试与其他班级展开心理与策略的生死博弈。

- **详细剧情**：
故事背景设定在日本政府创立的“高度育成高中”，该校升学率与就业率均为 100%，但实行彻底的“实力至上主义”。学生被分为 A 至 D 四个等级，班级点数决定待遇优劣，且点数可通过月考、体育祭及特殊的“特别考试”进行增减交易。主角绫小路清隆被分配至汇聚问题学生的 D 班，他隐藏了自己来自秘密实验机构“白色房间”的超凡实力，暗中观察并操控局势。
剧情核心围绕 D 班如何在资源匮乏、内部矛盾重重以及外班（尤其是 A 班坂柳有栖、C 班龙园翔等）的恶意打压下生存并晋升。每一场特别考试都不仅是智力的较量，更是人性、背叛与合作的试炼场。从无人岛生存测试到船上投票游戏，再到体育祭的阴谋反转，绫小路利用堀北铃音作为前台代理，联合轻井泽惠等关键人物，一次次化解危机，揭露学校背后的残酷真相，同时也在寻找自己作为“人”的意义。

## 关键问答 (FAQ)

**Q1: 《欢迎来到实力至上主义的教室》中的班级制度是如何运作的？**
A: 学校将学生分为 A、B、C、D 四个班级，每个班级拥有初始点数（货币）。班级点数高低直接决定学生的月度津贴、设施使用权等待遇。点数并非固定，而是通过每月的学业考试、体育竞技以及不定期举行的“特别考试”在班级间流动。最终目标是毕业时处于 A 班，只有 A 班学生才能获得进入顶尖大学或企业的推荐资格。

**Q2: 绫小路清隆为什么故意考进 D 班？**
A: 绫小路在进入高中前长期生活在名为“白色房间”的极端教育设施中，被培养成毫无感情的完美工具。他考入高度育成高中并故意控制分数进入 D 班，是为了逃避父亲的掌控，体验普通人的自由生活，并观察在不受“白色房间”规则束缚的环境下，人类的情感与成长是如何发生的。

**Q3: 特别考试（Special Exam）有什么特点？**
A: 特别考试是学校评价体系的核心，通常没有标准答案，规则复杂且充满陷阱。考试形式多种多样，包括无人岛生存、游轮投票、体育对抗、论文撰写等。其核心往往不仅考察智力或体力，更侧重考察学生的领导力、团队协作、心理博弈能力以及对规则的解读与利用能力，甚至鼓励背叛与欺诈。

**Q4: 堀北铃音和绫小路清隆是什么关系？**
A: 两人表面是同学，实际上是“操纵者”与“傀儡”的关系。绫小路认为堀北具备成为领导者的潜质但缺乏手段，因此躲在幕后为她出谋划策，让她站在台前处理班级事务，以此作为自己隐藏实力的掩护。随着剧情发展，堀北逐渐察觉真相并试图摆脱控制，两人关系转变为复杂的相互依存与理念碰撞。

**Q5: “白色房间”是什么地方？**
A: “白色房间”是绫小路清隆父亲主导的一个秘密教育实验项目。在这里，孩子从幼年起就接受极其严苛的智力、体能和心理训练，剔除不必要的情感，旨在培养出超越常理的“完美人类”。绫小路是该房间第四期唯一的成功幸存者，其他同期生均因无法承受压力而被淘汰。

**Q6: 为什么 D 班被称为“瑕疵品”聚集地？**
A: 因为 D 班的学生在入学考试中 scores 较低，或者在面试中表现出性格缺陷、社交障碍、过往不良记录等问题。学校将他们视为“次品”，给予最低的资源和待遇，意在通过残酷的竞争迫使他们自我淘汰或彻底改造。

**Q7: 龙园翔是如何统治 C 班的？**
A: 龙园翔是 C 班（后期为 B 班）的独裁式领袖。他不依靠民主或信任，而是通过暴力威胁、掌握把柄和恐惧来统治班级。他擅长策划恶毒的阴谋来陷害其他班级，尤其是针对 D 班。他的理念是“力量即正义”，直到被绫小路彻底击败后，其统治体系才开始动摇。

**Q8: 动画第三季对应原作的哪一部分？**
A: 动画第三季主要改编自原作轻小说的第 7.5 卷至第 11.5 卷的内容，涵盖了“新年特别考试”、“一之濑帆波的过去篇”以及“班级投票特别考试”等关键剧情，深入揭示了一之濑帆波的心理创伤及绫小路与之的互动。

## 名场面描述

1.  **无人岛考试的幕后反转**
    在无人岛生存特别考试中，D 班面临被退学的危机，且内部出现叛徒。当所有人以为 D 班束手无策时，绫小路清隆在幕后通过交换情报和操控点数，不仅精准揪出了叛徒，还反将一军，让企图陷害 D 班的龙园翔及其 C 班遭受重创。此役首次向观众（及部分角色）展示了绫小路深不可测的布局能力。

2.  **体育祭上的“假摔”与龙园的溃败**
    在体育祭的压轴项目“骑马战”中，龙园翔为了逼出 D 班的幕后黑手，对堀北铃音施以极度暴力的骑乘攻击。关键时刻，绫小路清隆不再隐藏，以压倒性的身体能力介入，不仅救下堀北，更在众目睽睽之下通过心理施压和武力威慑，彻底粉碎了龙园翔的自尊与统治逻辑，宣告了暴力统治的终结。

3.  **圣诞派对前的电梯告白与抉择**
    在圣诞特别考试前夕，轻井泽惠因过去的霸凌阴影再次崩溃，躲在教学楼电梯中。绫小路找到她，并非单纯安慰，而是冷静地分析局势并给予她“战斗”的理由。轻井泽决定不再逃避，主动请求成为绫小路的“专属道具”，两人在狭窄的电梯空间中确立了超越普通同学的深层羁绊，标志着轻井泽角色的彻底觉醒。
