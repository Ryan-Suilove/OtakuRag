# 关于我转生变成史莱姆这档事 (Tensei Shitara Slime Datta Ken)

## 基础信息
- **中文名**：关于我转生变成史莱姆这档事
- **日文名**：転生したらスライムだった件
- **类型**：异世界转生、奇幻、冒险、建国、喜剧
- **首播年份**：2018 年（第一季）
- **制作公司**：8bit
- **原作**：伏濑（著作）、みっつばー（插画）的同名轻小说系列

## 主要角色

### 角色名：利姆鲁·特恩佩斯特 (Rimuru Tempest)
- **身份**：主角 / 魔国联邦盟主 / 魔王
- **声优**：冈咲美保（日）/ 丰口惠美（前世：三上悟）
- **人物设定**：前世为日本上班族三上悟，被刺死后转生为异世界的史莱姆。性格温和包容，兼具领导者的决断力与老好人的随和。拥有独特技能“大贤者”（后进化为“智慧之王”）和“捕食者”，能吞噬万物并解析模仿其能力。致力于建立人类与魔族共存的理想国家。
- **经典台词**：“既然来到了这个世界，那就随心所欲地活下去吧。”

### 角色名：红丸 (Benimaru)
- **身份**：利姆鲁的得力部下 / 鬼人族首领 / 魔国联邦大将
- **声优**：古川慎
- **人物设定**：原为大鬼族部落的幸存者，被利姆鲁命名后进化为鬼人族。拥有极高的战斗天赋和统率力，性格忠诚且沉稳，是利姆鲁最信任的战力之一。擅长火焰系魔法与剑术，负责魔国联邦的军事指挥与治安维护，对利姆鲁有着绝对的信赖。
- **经典台词**：“只要您下令，无论何处我都会为您斩开一条血路。”

### 角色名：朱菜 (Shuna)
- **身份**：利姆鲁的部下 / 鬼人族公主 / 内政总管
- **声优**：千本木彩花
- **人物设定**：红丸的妹妹，同样因利姆鲁的命名而进化。外表美丽温柔，擅长料理、裁缝及内政管理。她是魔国联邦文化发展的核心推动者，将日本的文化（如温泉、和食）引入异世界。虽然平时温文尔雅，但在保护利姆鲁或国家利益时也会展现出强大的法术战斗力。
- **经典台词**：“利姆鲁大人，这道新开发的料理，请您务必尝尝看。”

### 角色名：维鲁德拉·特恩佩斯特 (Veldora Tempest)
- **身份**：利姆鲁的挚友 / “风暴龙” / 四大龙种之一
- **声优**：前野智昭
- **人物设定**：被封印在洞穴中三百年以上的真龙，性格狂妄自大却意外地单纯像个孩子。与利姆鲁相遇后成为其第一个朋友，并将名字借给利姆鲁（故利姆鲁姓特恩佩斯特）。拥有毁灭性的魔力，是利姆鲁前期的最强靠山，后期常以人类形态在图书馆看书或喝酒。
- **经典台词**：“哼，吾之名为维鲁德拉！记好了，凡人！”

## 剧情概要

- **一句话简介**：普通上班族转生为史莱姆，凭借独特能力收服众多种族，在异世界建立人魔共存理想国的史诗故事。

- **详细剧情**：
故事始于日本上班族三上悟在街头为保护后辈而遇刺身亡。当他醒来时，发现自己转生到了充满魔法与魔物的异世界，并变成了最弱魔物“史莱姆”。在封印之地偶遇被囚禁三百年的“风暴龙”维鲁德拉，两者结为挚友，利姆鲁获得了龙的力量与名字。
为了履行对维鲁德拉的承诺并寻找生存之道，利姆鲁开始探索世界。他利用独特技能“捕食者”和“大贤者”不断吞噬魔物、解析技能并进化。途中，他解救了被人类迫害的哥布林、牙狼族及大鬼族余孽，并通过“命名”赋予他们力量，使其进化为高阶种族。
随着伙伴增多，利姆鲁建立了魔国联邦（特恩佩斯特），推行人类与魔族平等共存的政策。然而，这一举动触动了人类王国与激进魔族势力的利益。从对抗猪头帝引发的战争，到应对人类诸国的围剿，再到觉醒
