# 🗓 五个月 AI 岗求职全流程行动计划（最终版）

---

## 🟦 第 1 个月：基础巩固 + 微调项目成型
### 🎯 核心目标
打牢 LLM 微调和 Python / PyTorch 基础，保证项目可展示。

### 📌 任务列表
1. 完成“QQ语气模仿 Bot”微调项目（LoRA / Q-LoRA）
   - 数据集清洗、格式化
   - 微调 DeepSeek/Qwen
   - 实验不同 rank/alpha 对比
2. 刷基础算法题（数组、链表、哈希表、栈、队列）
   - 每周 4–6 题，LeetCode easy + 部分 medium
3. 学习 Transformer、Attention、LoRA/QLoRA 原理

### 📤 输出物
- 训练曲线、LOSS 对比图、生成结果对照
- README 文档
- 推理 Demo（FastAPI / Streamlit）

---

## 🟩 第 2 个月：毕设项目 + 理论提升
### 🎯 核心目标
完成 AutoLayout 核心算法实现，掌握训练全流程。

### 📌 任务列表
1. AutoLayout 毕设核心算法（只做可展示的基础功能）
   - 文本指令解析 (LLM)
   - 图结构调整基础算法
   - Python API 简单 demo
   - 包装策略：强调算法思路、核心模块实现、可扩展性
2. 补齐 PyTorch Lightning / HF Trainer 使用经验
3. 刷中等算法题（滑动窗口、栈、队列、二叉树基础）
   - 每周 5–8 题
4. 深入理解 LLM 训练原理和优化技巧

### 📤 输出物
- AutoLayout demo + 文档
- 核心算法实验记录

---

## 🟧 第 3 个月：BitNet 低比特推理 + 部署落地
### 🎯 核心目标
完成模型量化、推理优化，具备落地能力。

### 📌 任务列表
1. BitNet 低比特推理项目
   - 1.58bit / 4bit / 8bit 性能对比
   - 显存占用对比
   - 精度对比
2. 学习 GPU 推理优化、KV Cache、Tensor 并行等概念
3. FastAPI 部署微调模型 Demo
4. 刷中等/较难算法题（树、图、动态规划）
   - 每周 6–10 题

### 📤 输出物
- 推理性能报告
- 可访问 API Demo
- 实验日志

---

## 🟨 第 4 个月：简历打磨 + 系统设计 + 模拟面试
### 🎯 核心目标
让项目可落地，可讲可展示；熟悉面试问答体系。

### 📌 任务列表
1. 整理项目亮点：背景 / 技术难点 / 实验结果 / 优化方案 / 遇到问题及解决
2. 学习 AI 系统设计题：RAG 检索系统、部署亿级 token LLM、KV Cache 分片
3. 模拟面试（项目深问 + LLM 理论 + 算法题 + 系统设计）
4. 刷精选算法题
   - 每周 8–12 题
   - 开始计时模拟面试环境

### 📤 输出物
- 项目简历可讲版本
- 系统设计图和讲解笔记
- 模拟面试反馈总结

---

## 🟥 第 5 个月：投递 + 面试冲刺
### 🎯 核心目标
进入实际面试流程，获取 offer。

### 📌 任务列表
1. 每周投递 30–50 份 AI 岗位简历（算法岗 / 开发岗）
2. 每周 2–3 场模拟面试，针对反馈优化
3. 持续补充薄弱环节（算法、理论、工程）
4. 学习求职流程和注意事项：简历筛选、HR 电话面试、技术面试、Offer 谈判

### 📤 输出物
- 面试安排表
- 投递记录表
- 面试笔记与总结

## 📌 刷题细化计划
| 月份 | 题型 | 数量 | 难度 |
|------|------|------|------|
| 第1个月 | 数组、链表、哈希表、栈、队列 | 每周4–6题 | easy/medium |
| 第2个月 | 树、滑动窗口、双指针、回溯基础 | 每周5–8题 | medium |
| 第3个月 | 图论、拓扑排序、并查集、DP基础 | 每周6–10题 | medium+/hard |
| 第4个月 | 综合题：复杂DP、树/图混合、字符串匹配 | 每周8–12题 | medium+/hard |
| 第5个月 | 大厂面经题、高频算法题 | 每周10–15题 | hard |
